Entity
Definition 
Physical Location is an area bounded by parameters such as Latitude and
Longitude, Altitude, Depth, etc.

Physical Location Name 
Physical Location Name is name given to the physical location of the property,
e.g. Rockefeller Center.

Latitude Value 
Latitude Value is the value on a grid of the globe that locates the point
horizontally East to West.
 No No No
Longitude Value 
Longitude Value is the value on a grid of the globe that locates the point
Vertically north to south.
 No No No
Altitude Value 
Altitude Value is the vertical distance or height measured upward from sea
level to a point in the atmosphere.
 No No No
Altitude Mean Sea Level Value Altitude Mean Sea Level Value is the average sea level height. No No No
Horizontal Accuracy Value 
Horizontal Accuracy Value defines the horizontal location for a property, e.g.
1.000 feet from the hydrant.
 No No No
Vertical Accuracy Value 
Vertical Accuracy Value is the height of a location, for example 4,000 foot
height up the mountain.
 No No No
Travel Direction Description 
Travel Direction Description provides the instructions for driving to a
particular location.
 No No No
Location Address Identifier 
Location Address Identifier is assigned by the insurance carrier to uniquely
identify a physical site that generally includes street number, street, city,
state/province, postal code, and country.  This location can be a structure that
contains a home, commercial building, vehicle or items to be insured, as well as
a site where a party can recieve mail.
 No No Yes

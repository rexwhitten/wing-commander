Day and Time Group Code represents a general and relative set of time frames.
 Examples:"WDWH" for Week Days during Work Hours, "WE" for Weekends.
Day Code Code The code which represents the day of a week. Example: MON= Monday, SAT=Saturday.
Day Type Code Code 
Day Type Code is a code that can be used to differentiate between work days,
weekend days, and holidays.
Deductible Basis Code Code 
Deductible Basis Code identifies whether the related Deductible Value is an
amount or a percentage or number of days, etc. that will be used to determine
what an insured is expected to pay before reimbursement from the insurance
company begins.
Deductible Type Code Code 
Deductible Type Code represents the kind of deductible.  Examples: per
occurrence, per person.
Event Sub Type Code Code Event Sub Type Code represents a lower level event category.
Event Type Code Code 
Event Type Code represents the kind of an event.  Examples:  life event,
business event, policy event or loss event.
Gender Code Code 
Gender Code represents the sex of the individual. Examples: F for female, M for
Male, U for Unknown.
Geographic Location Type Code Code 
Geographic Location Type Code represents a category of a bounded area defined
by nature, by an external authority (such as a government) or for an internal
business purpose. Used to identify a location in space that is not a structured
address.  Examples: country, city, continent, postal area, risk area.
Industry Catastrophe Code Code 
Industry Catastrophe Code is a Code assigned by a regulatory agency to identify
 a single major loss occurring on a specified date or to a single loss incurred
over a number of days.
Industry Code Code Industry Code represents an Industry.
Industry Type Code Code 
Industry Type Code represents an Industry category. Examples: Insurance,
Banking, Pharmaceutical, Utility
Insurable Object Type Code Code 
Insurable Object Type Code represents a category of an item figuring  in an
insurance coverage or policy, having commercial or exchange value that is owned
by a business, institution, or individual or adds to one's wealth or estate.
These objects can be real (land and attachments) and personal (movable effects
not attached to land).
Insurance Type Code Code 
Insurance Type Code represents the Insurance Type.  Examples: Direct, Assumed,
Ceded

Legal Classification Code Code

perform as agreed in the contract.
Limit Basis Code Code 
Limit Basis Code represents the basis on which to specify the maximum extent of
coverage that will be provided by an insurance policy.  Example: Amount,
Percentage, Days.
Limit Type Code Code 
Limit Type Code represents the kind of limit that will be provided by an
insurance policy. Examples: Per Occurrence, Per Person.
Line Of Business Code Code 
Line Of Business Code represents a type of major Product Line.  Examples: 
automobile and home owners insurance.
Locality Code Code 
Locality Code represents a location category used by a party. Examples: home,
work, second residencex
Location Code Code Location Code represents an externally assigned identification for a location.
Month Code Code A standard 3 position abbreviation for each month of a year.
Organization Type Code Code 
Organization Type Code represents an organization category. Values: I =
Internal or E = External.
Party Role Code Code 
Party Role Code is an abbreviation of name of the role that a Party plays with
respect to another entity.
Party Type Code Code 
Party Type Code is an abbreviation that defines the kind of Party.  Example:
Person, Organization, Grouping.
Postal Code Code 
Postal Code represents an alphanumeric code assigned to groups of addresses by
various national postal systems throughout the world to facilitate mail delivery
Rating Territory Code Code 
Rating Territory Code represents a geographic area that is used to determine
insurance rates.
Relationship Code Code Relationship Code represents a relationship that ties entities together.
Relationship Type Code Code 
Relationship Type Code represents the kind of relationship that links one
entity to another,  Example:  Business, Family, Reporting,
Staff Classification Code Code 
Staff Classification Code defines the kind of Staff as Employee or Contractor,
the eligible benefits, and level of a staff person.
State Code Code 
State Code is a two character representation of a U.S. state or Canadian
Province.
Status Code Code 
Status Code represents the state of the related entity at a point in time.
Examples: A = Active, P = Pending, C = Cancelled, T = Terminated.
Work Site Type Code Code 
Work Site Type Code defines the kind of workplace.  Example: Office Building,
Plant, Farm.

Name String 
Name is the wording used to identify an Organization, Group or full name of an
Individual.
Day Name Name Day Name is the name of a day in a week.
Long Name Name 
Long Name is the full name of any object, such as the entire legal name of an
Organization, Group or full name of an Individual.
Month Name Name Month Name is full English name of a Month.
Prefix-Suffix Name Name

Jr., Sr., III, M.D., PhD.
Short Name Name 
Short Name is a brief name used for an object or a partial name of an
individual, such as first, last or middle name.
Variable Value String 
Variable Value is an amount, code or any other type of data that can contain
syntax (such as hyphen separators) and delimiters as needed for interpretation
and display.
Company Claim Number Variable Value 
Company Claim Number is the number or alpha-numeric assigned to a claim by the
company.
Company Subclaim Number Variable Value 
Company Subclaim Number identifies a claim within a claim. It can be an
alpha-numeric value.
Deductible Value Variable Value 
Deductible Value is the actual value of a monetary amount, percentage or time
period.  Examples: $500, 10%, 90 days
Latitude Or Longitude Variable Value 
Latitude or Longitude is a metric expressed in a common format to identify a
position on the globe.  The values can be expressed as Degree, Minute, Second,
Decimal.
Limit Value Variable Value 
Limit Value is the actual limit amount or percentage that will be provided by
an insurance policy.
Policy Form Value Variable Value 
Party Form Value is the set of characters used to identify an attachment to a
property and casualty policy owned by a party that makes it operative.
Policy Number Value Variable Value 
Policy Number Value is a value assigned to uniquely identify an insurance
policy that can be expressed in letters, numbers or characters
Variable Identifier Value Variable Value 
Variable Identifier Value is an externally assigned value or identifier.
 Example: phone number, company site number
Long Variable Value Variable Identifier Value 
Long Variable Value is text that can contain syntax or delimiters as needed for
interpretation and display, such as a web link. Example:
http://www.naic.org/store_jir.htm
Medium Variable Value Variable Identifier Value 
Medium Variable Value is a number, date, or text that may contain syntax or
delimiters as needed for interpretation and display. Example: (212) 645-2387,
extension 437

Aggregate Limit Amount 
Aggregate Limit Amount is a limit of coverage that specifies the maximum amount
(or other limit expression such as number of days) to be paid regardless of any
other factor.

Maximum Per Claim Amount 
Maximum Per Claim Amount is the Limit that can be paid for any single event for
which a claim is reported.   This amount is inclusive of all coverage provided
by an insurance policy.
